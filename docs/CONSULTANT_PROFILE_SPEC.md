# 컨설턴트 프로필 명세

> 이 문서는 컨설턴트 프로필 폼의 카테고리, 설명, 선택지를 정의합니다.
> 수정 후 개발에 반영됩니다.

---

## 1. 소속

**설명:** 현재 소속된 회사 또는 기관명을 입력해주세요. 소속이 없는 경우 '프리랜서'로 입력합니다.

**입력 방식:** 텍스트 입력 (1~50자, 필수)

---

## 2. AI 훈련 가능 산업

**설명:** AI 훈련/코칭을 수행할 수 있는 산업을 선택해주세요.

**입력 방식:** 다중선택 (배지)

| 선택지 |
|--------|
| 제조업 |
| 서비스업 |
| 유통/물류 |
| IT/소프트웨어 |
| 금융/보험 |
| 건설/부동산 |
| 의료/헬스케어 |
| 에너지/환경 |
| 농업/식품 |
| 교육 |
| 공공/정부 |

---

## 2-1. 선호 세부 업종

**설명:** 더 구체적인 업종을 입력해주세요. 세부 업종을 추가하면 더 정확한 매칭이 가능합니다. (예: 반도체, 디스플레이, 자동차 부품, 식품 등)

**입력 방식:** 태그 입력 (선택, 자유 텍스트)

---

## 3. AI 적용 가능 업무

**설명:** AI를 활용해 개선할 수 있는 업무 영역을 선택해주세요.

**입력 방식:** 다중선택 (배지)

| 선택지 |
|--------|
| 생산/제조 |
| 품질관리 |
| 설비/시설관리 |
| 연구개발(R&D) |
| 구매/SCM |
| 물류/유통 |
| 영업/마케팅/홍보 |
| 인사/총무 |
| 기획/전략 |
| 법무/계약 |
| 재무/회계 |
| IT/시스템 |
| 고객서비스 |
| 디자인/콘텐츠 |
| 문서/보고서 작성 |
| 데이터 수집/분석 |

---

## 4. 교육 대상 수준

**설명:** 교육 가능한 학습자의 AI 활용 수준을 선택해주세요.

**입력 방식:** 다중선택 (배지)

| 값 | 라벨 | 설명 |
|----|------|------|
| BEGINNER | 입문 | AI 개념 이해, 비사용자 대상 기초 교육 |
| INTERMEDIATE | 실무 | AI 도구 활용, 현업 적용, 프롬프트 설계 |
| ADVANCED | 심화 | 워크플로우 구축, 자동화 도구 제작 |
| LEADER | 리더 | AI 전략 수립, 조직 내재화 |

---

## 5. 선호 교육 방식

**설명:** 주로 활용하는 교육/코칭 방식을 선택해주세요.

**입력 방식:** 다중선택 (배지)

| 값 | 라벨 |
|----|------|
| PBL | PBL (프로젝트 기반) |
| WORKSHOP | 워크숍 |
| MENTORING | 코칭/멘토링 |
| LECTURE | 강의 |
| HYBRID | 혼합형 |

---

## 6. 보유 역량

**설명:** 본인이 보유한 핵심 역량을 모두 선택해주세요.

**입력 방식:** 다중선택 (배지)

| 선택지 |
|--------|
| 생성형 AI 활용 (ChatGPT, Claude, Gemini 등) |
| 프롬프트 엔지니어링 |
| AI 코딩 도구 (클로드코드, 코덱스, 안티그래비티, 커서AI 등) |
| 협업 도구 AI 활용 (Notion AI, Copilot for M365, Slack AI 등) |
| AI 디자인 도구 활용 (Midjourney, Canva AI, Figma AI 등) |
| 업무 자동화 (RPA/노코드) |
| 워크플로우 설계 |
| 데이터 수집/정제 |
| 데이터 시각화 |
| 문서/보고서 작성 |
| 품질/통계 분석 |
| 프로세스 개선 |
| 변화관리/내재화 |
| AI 전략/거버넌스 |
| 보안/컴플라이언스 |

---

## 7. AI 교육/컨설팅 경력

**설명:** AI 관련 교육 또는 컨설팅 경력을 입력해주세요.

**입력 방식:** 숫자 입력 (0~50년)

---

## 8. 경력 사항

**설명:** 이전 직장, 근무 기간, 담당 업무 등 경력을 작성해주세요.

**입력 방식:** 텍스트 입력 (50~2,000자)

---

## 9. 강의/컨설팅 포트폴리오

**설명:** 진행했던 강의, 워크숍, 컨설팅 등을 작성해주세요. 자세히 작성할수록 매칭 건수가 많아집니다.

**입력 방식:** 텍스트 입력 (50~2,000자)

---

## 10. 강점/제약

**설명:** 본인의 강점과 제약사항을 작성해주세요.

**입력 방식:** 텍스트 입력 (50~2,000자)
